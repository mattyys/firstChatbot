## Clonar proyecto

Elije una carpeta de preferencia, abre la terminal  ejecuta el siguiente comando.

```
git clone https://github.com/mattyys/firstChatbot.git
```

Accede a la carpeta descrgada, instala dependencias y corre el proecto.

```
cd firstchatbot
npm install
npm run dev
```
